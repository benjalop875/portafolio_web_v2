## 🛠️ Tech Stack & Features

- **Framework**: [Astro 6.0](https://astro.build/) - Rendimiento de vanguardia y arquitectura de islas.
- **Styling**: [Tailwind CSS](https://tailwindcss.com/) - Diseño responsivo con sistema de diseño personalizado (Material Design 3 inspired).
- **I18n**: Soporte completo para **Español** e **Inglés**.
- **Dark Mode**: Selector de temas inteligente integrado (Light/Dark/System).
- **Icons**: Sistema modular de componentes SVG personalizados.
- **Scroll Spy**: Navegación dinámica que resalta la sección activa.
- **Contact Form**: Integración real con **Formspree** para recepción de mensajes.

```bash

# Construir para producción / Build for production
npm run build
```
